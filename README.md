# A&N Enterprise Website

This is a premium Next.js website for A&N Enterprise (AC Repair & Installation).
Built with Next.js 15, Tailwind CSS v4, and TypeScript.

## Features
- **Modern Design**: Premium aesthetics with smooth animations/interactions.
- **Responsive**: Fully optimized for Mobile, Tablet, and Desktop.
- **SEO Optimized**: Semantic HTML and fast loading speeds.
- **Contact Integration**: Integration with WhatsApp for direct booking.

## Pages
- **Home**: Landing page with Hero, Services, and Testimonials.
- **About Us**: Company mission and details.
- **Services**: Detailed service breakdown.
- **Gallery**: Before/After work portfolio.
- **Contact**: Request form and location details.

## Getting Started

1. **Install Dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the site.

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Customization
- **Images**: Add your own project photos to `public/images/`.
- **Contact Info**: Update phone numbers and links in `components/layout/Footer.tsx` and `app/contact/page.tsx`.
